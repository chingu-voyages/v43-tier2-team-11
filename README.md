## Usage

### Prerequisites

Install npm  
`npm install npm@latest -g`

### Getting Started

1. Clone the repo  
   `git clone https://github.com/chingu-voyages/v43-tier2-team-11.git`
2. Install NPM packages  
   `npm install`
